# Dishwasher

**Category:** Appliance
**Covers:** Common dishwasher error codes and troubleshooting (Bosch, Whirlpool, Maytag, GE, Samsung, LG)

## Error Codes

### Bosch Dishwashers
| Code | Meaning | Fix |
|------|---------|-----|
| E15 | Water in base pan (float triggered) | Tilt dishwasher forward 45° to drain base pan; check hose connections; power cycle |
| E22 | Filter clogged | Remove and clean drain filter at bottom of tub |
| E24 | Drain issue — won't drain | Check drain hose for kinks; clean filter; clear garbage disposal if shared |
| E25 | Drain pump blocked | Remove bottom rack and spray arm; clean pump area |
| E09 | Heating element failure | Professional service required |
| E01 | Control board error | Power cycle; if persists, control board issue |

### Whirlpool / Maytag
| Code | Meaning | Fix |
|------|---------|-----|
| F6E1 / F6E2 | Water supply issue | Check water supply valve under sink |
| F7E1 | Heating circuit issue | Professional service |
| F8E1 / F8E4 | Drain issue | Clean filter and check drain hose |
| F9E1 | Drain pump failure | Clean pump; if persists, pump replacement |

### Samsung / LG
| Code | Meaning | Fix |
|------|---------|-----|
| 1E / IE | Water supply | Check supply valve and inlet hose |
| 5E / OE | Drain issue | Clean filter; check drain hose and garbage disposal |
| 7E | Water level sensor | Power cycle |
| LC / LE | Leak detected | Check connections; tilt to drain base if needed |

## Diagnostic Steps

1. **Note any error code or flashing lights** on the display
   - *Camera cue: Some models flash LED patterns instead of alphanumeric codes (count flashes)*

2. **Check door latch** — dishwasher won't start unless door is fully closed
   - *Camera cue: Push door firmly until latch clicks. Some models require pushing button + closing*

3. **Check for standing water in the bottom of the tub**
   - *Camera cue: Small puddle (1 inch) is normal. Several inches = drain problem*

4. **Clean the drain filter** — bottom of the tub, center
   - *Camera cue: Circular filter that twists counterclockwise to remove. Rinse under warm water*

5. **Check the drain hose** under the sink
   - *Camera cue: Should have a high loop (or air gap) and connect to garbage disposal or drain tailpiece*

6. **Check if garbage disposal knockout plug was removed** (new installations)
   - If disposal was recently installed, the dishwasher drain knockout must be removed first

## Common Issues

| Problem | Cause | Fix |
|---------|-------|-----|
| Won't start | Door latch, child lock, or delayed start | Ensure door latched. Check child lock (hold button 3 sec). Check delay start setting |
| Won't drain | Clogged filter, kinked drain hose, or disposal blockage | Clean filter; check drain hose; run garbage disposal |
| Standing water / E15 (Bosch) | Water in base pan | Tilt dishwasher forward to drain. Check for leaking hoses |
| Dishes not clean | Filter dirty, spray arms clogged, low water temp | Clean filter and spray arm holes; check water heater is set to 120°F |
| Dishes not dry | Rinse aid empty, or heated dry setting off | Refill rinse aid; check drying settings |
| Leaking from door | Door seal worn or debris on gasket | Clean door seal; inspect for tears |
| Grinding noise | Debris in pump area or broken spray arm | Remove bottom rack, check pump area and spray arms |

## Safety Notes
- **Disconnect power** before accessing internal components (pump, heater element)
- Be careful of **sharp items** when cleaning the filter (broken glass often collects there)
- Bosch E15: water in base = tilt forward, but disconnect power first to avoid shorting the control board
