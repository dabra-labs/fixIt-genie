# Washing Machine

**Category:** Appliance
**Covers:** Error codes and troubleshooting across major brands (LG, Samsung, Whirlpool, Maytag, GE)

## Error Codes by Brand

### LG Washing Machines
| Code | Meaning | Fix |
|------|---------|-----|
| IE / E4 | Water supply issue — not filling | Check supply valves are open; check inlet hose for kinks; clean inlet filter screens |
| OE | Drain issue — not draining | Check drain hose for kinks; clean drain pump filter (bottom front); check standpipe height |
| UE | Unbalanced load | Redistribute clothes evenly; reduce load size; check machine is level |
| dE / dE1 | Door not latched | Clean door gasket; check for obstructions; door switch may need replacement |
| LE | Motor overload | Reduce load; unplug 30 min and retry; if persistent, motor/rotor sensor issue |
| PE | Water level sensor | Power cycle; if persists, pressure hose or sensor needs replacement |
| tE | Water temperature sensor | Power cycle; check heater element |
| CE | Current error — motor overloaded | Reduce load; power cycle |

### Samsung Washing Machines
| Code | Meaning | Fix |
|------|---------|-----|
| 4E / 4C | Water supply | Check supply valves; clean inlet filters |
| 5E / 5C / SE | Drain issue | Clean pump filter; check drain hose |
| UE / U6 | Unbalanced | Redistribute load |
| dE / dC | Door error | Check door latch and gasket |
| 3E / 3C | Motor error | Reduce load; power cycle |
| E3 | Water level error | Power cycle; check pressure hose |

### Whirlpool / Maytag (F-codes)
| Code | Meaning | Fix |
|------|---------|-----|
| F21 | Long drain — pump clog | Clean pump filter; check drain hose |
| F01 | Primary control failure | Power cycle; if persists, control board |
| F20 | No water detected | Check supply valves and hoses |
| F51 | Motor rotor position sensor | Power cycle; may need sensor replacement |
| SUD | Too much suds | Run rinse cycle; use HE detergent only |

## Diagnostic Steps

1. **Note the error code** displayed on the panel
   - *Camera cue: Error code appears on digital display — may be letters + numbers, or flashing lights*

2. **Power cycle first** — unplug for 60 seconds, then plug back in
   - Clears many temporary control board errors

3. **Check door/lid** — must be fully closed and latched
   - *Camera cue: Front loader: check gasket for debris. Top loader: lid switch tab must engage*

4. **Check water supply valves** behind the machine — both fully open
   - *Camera cue: Handles should be parallel to hose (open position)*

5. **Check drain hose** — shouldn't be kinked or inserted too far into the standpipe
   - *Camera cue: Drain hose should only go 6–8 inches into the standpipe, and standpipe should be 34–96 inches high*

6. **Clean the drain pump filter** (front loaders — small door at bottom front)
   - *Camera cue: Round cap behind small access panel at front bottom. Have towels ready — water will spill*

## Common Fixes

| Problem | First Check | Second Check |
|---------|------------|--------------|
| Won't fill | Supply valves open? | Inlet filter screens clean? |
| Won't drain | Drain hose kinked? | Pump filter clogged? |
| Won't spin | Load balanced? | Is drum overloaded? |
| Leaking | Door gasket tears? | Drain hose connection loose? |
| Loud noise during spin | Load unbalanced? | Machine level? Shipping bolts removed? |
| Not cleaning well | Using HE detergent? | Drum clean cycle needed? |

## Safety Notes
- **Always unplug** before accessing internal components (pump filter, hoses)
- Water + electricity = danger — mop up water before working near outlets
- The drain pump filter may release water when opened — **have towels ready**
- Front loader door gaskets harbor mold — wipe dry after each use, leave door ajar
