# LG Refrigerator

**Category:** Appliance
**Covers:** LG refrigerator error codes, cooling issues, ice maker, water dispenser, compressor

## Error Codes

| Code | Meaning | Fix |
|------|---------|-----|
| OFF / 0FF | Likely demo/showroom mode | Say the display text first. Explain that the fridge may be in demo mode, which leaves the display on but disables cooling. If the exact model is known, look up the model-specific exit sequence before giving button instructions |
| FF only (partial read) | Ambiguous partial display | Do not guess. Ask for a tighter, steadier shot of the full water-dispenser/control panel to distinguish OFF/0FF from Er FF |
| Er IF | Ice maker fan blocked or frozen | Remove all ice from ice maker; power cycle; if persists, fan motor needs replacement |
| Er FF | Freezer evaporator fan failure | If it clearly shows Er FF, defrost manually (hold both temp buttons 3–5 sec) to unblock the fan; if it persists, fan replacement |
| Er CF | Condenser fan blocked or failed | Clean condenser area at back/bottom; check fan blade for debris; power cycle |
| Er dF | Defrost heater or sensor failure | Run manual defrost (hold both temp buttons 3–5 sec); if error returns, defrost heater replacement needed |
| Er rF | Fan speed sensor | Power cycle; if persists, professional service |
| Er CO | Communication error | Power cycle; check wiring harness connections; may need control board |
| Er FS | Freezer sensor | Power cycle; sensor replacement if persists |
| Er IS | Ice maker sensor | Power cycle; ice maker sensor replacement |
| Er SS | Sub-cooling sensor | Professional service |
| CL | Child lock active | Hold LOCK button 3 seconds to toggle |
| dH | Defrost heater issue | Run manual defrost; if error persists, heater or sensor needs replacement |

## Diagnostic Steps

1. **Check temperature display first**
   - *Camera cue: Display panel on front door or inside at top. If the panel says `OFF` or `0FF`, treat that as a likely demo/showroom mode clue before deeper cooling troubleshooting. If you can only make out `FF`, ask for a closer view before deciding what it means*

2. **If the panel does not say `OFF` or `0FF`, listen for the compressor** — should cycle on and off
   - *Camera cue: Compressor is at back bottom — humming sound means it's running*

3. **Check condenser coils** on back or bottom
   - *Camera cue: Dusty/grey coils at the back = poor heat dissipation = warm fridge. Clean with vacuum brush*

4. **Check door seals all around** — should be airtight
   - *Camera cue: Paper test: close a sheet of paper in the door — if it slides out easily, seal is weak*

5. **Check inside air vents** — shouldn't be blocked by food containers
   - *Camera cue: Cold air vents on back wall inside — leave a few inches of clearance*

6. **Check for ice buildup on freezer back wall**
   - *Camera cue: Thick frost/ice on interior back wall = defrost system problem*

## Common Issues

| Symptom | Cause | Fix |
|---------|-------|-----|
| Display says `OFF` or `0FF` and fridge is not cooling | Likely demo/showroom mode | Call out the display text first. Explain that the fridge may be in demo mode. If the model number is available, use it to look up the exact exit button sequence before guessing |
| Display shows only `FF` | Ambiguous read | Ask for a closer shot of the full water-dispenser/control panel and tell the user to hold it steady. A partial `FF` can actually be `OFF`/`0FF` or part of `Er FF` |
| Fridge not cooling, freezer fine | Evaporator fan blocked or failed, damper stuck | Hold door button to test if fan runs. Ice on back wall = defrost issue; run manual defrost |
| Neither fridge nor freezer cooling | Compressor not running or refrigerant leak | Check start relay (small box on compressor side) — rattle when shaken = replace (~$10 part) |
| Ice maker not making ice | Freezer too warm, water supply issue, or arm in UP position | Check freezer is 0°F. Check water line isn't kinked. Check ice maker arm is DOWN. Replace filter if >6 months old |
| Loud noise | Ice hitting evaporator fan (defrost needed) or condenser fan debris | Defrost the freezer. Clean condenser area |
| Water dispenser not working | Clogged filter, frozen water line, or door switch | Replace filter (>6 months old). Check filter is fully seated. Thaw water line if frozen |
| Water inside fridge | Drain tube blocked (ice or debris) | Pour warm water down defrost drain tube (usually behind back panel inside freezer) |
| Fridge making ice/frost inside | Door seal issue or high humidity | Test door seal with paper test. Reduce door open frequency |

## Special LG Features

**Manual Defrost:**
Press and hold both the fridge and freezer temperature buttons simultaneously for 3–5 seconds to initiate a manual defrost cycle.

**Smart Diagnosis:**
Place your phone against the LG logo on the fridge front and call LG support (1-800-243-0000). The fridge transmits a series of audio tones that LG can decode to identify the fault code.

**Filter Reset:**
After replacing the water filter, press and hold the FILTER button for 3 seconds to reset the indicator light.

**LG Linear Compressor:**
LG Linear Compressor models are very quiet. If you hear loud knocking or banging, the compressor may be failing — call LG service.

## Safety Notes
- **Unplug before cleaning condenser coils** or accessing internal components
- **Never use sharp objects to chip ice** — can puncture refrigerant lines
- If you smell a chemical odor, refrigerant may be leaking — ventilate the room and call a technician
- Refrigerant leaks require EPA-certified technicians to repair
